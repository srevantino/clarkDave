**A-SYS_clark** is developed and managed by **Advance Systems 4042**. This site documents the A-SYS_clark Windows utility (based on the open-source WinUtil project).

## Running A-SYS_clark

* Start PowerShell **as Administrator**.
* Run the launch command provided by Advance Systems 4042, for example:

```
irm 'https://clark.advancesystems4042.com/?token=YOUR_TOKEN' | iex
```

Update the URL to match your organization’s current deployment.

> [!NOTE]
> Many pages below still refer to “Winutil” from upstream documentation. Treat those as describing A-SYS_clark behavior unless noted otherwise.
